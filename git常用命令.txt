查看分支：git branch
创建分支：git branch name
切换分支：git checkout name
创建+切换分支：git checkout Cb name
合并某分支到当前分支：git merge name
删除分支：git branch Cd name
1. 要查看远程库的信息 使用 git remote
2. 要查看远程库的详细信息 使用 git remote Cv
使用命令 git push origin master 推送到服务器
在dev分支上做开发，就必须把远程的origin的dev分支到本地来，于是可以使用命令创建本地dev分支：git checkout Cb dev origin/dev
git pull把最新的提交从origin/dev抓下来
